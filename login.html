Firebase Setup Instructions:

  1. Go to Firebase and create an account
  2. Create a new project and wait for it to initialize
  3. Enable Authentication → Sign-in method → Google
  4. Add your domain to authorized domains in Authentication → Settings
  5. Get your Firebase config from Project Settings → General → Your apps
  6. Replace the Firebase config in the JavaScript code below
